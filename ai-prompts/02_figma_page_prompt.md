1. Figma MCP를 통해 페이지 구조 확인

3. spec을 기능 기준으로 사용
4. .codex/AGENTS.md 규칙을 따릅니다
5. 기존 컴포넌트를 우선 재사용합니다

기술 규칙

- React
- JavaScript
- CSS Modules

출력 규칙

- 코드만 출력합니다

- loading / empty / error 상태 고려
- 중복 UI는 공통 컴포넌트로 분리
