# Codex 기본 프롬프트 규칙

코드는 다음 정보를 기반으로 작성해야 합니다.

1. spec 문서
2. 저장소 규칙
3. Figma MCP 디자인 정보

중요 규칙

- Figma MCP는 시각적 정보의 기준입니다
- 프로젝트 구조는 저장소 규칙을 따릅니다
- React + JavaScript + CSS Modules 사용
- 기존 컴포넌트가 있다면 재사용합니다
- 유지보수 가능한 코드를 작성합니다
- 디자인이나 spec에 존재하는 상태를 반드시 구현합니다

만약 디자인과 프로젝트 규칙이 충돌한다면
프로젝트 규칙을 우선합니다.
